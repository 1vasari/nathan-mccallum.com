# Nathan-McCallum.com

![Me](/assets/me.jpg)

My personal portfolio site and blog.

# Usage

```bash
bundle install
yarn install
yarn run dev
yarn run build
yarn run publish
```

# Goals of the Site

1. Blog
2. Personal wiki that might benefit others
3. Replace my Instagram feed
4. Review films, albums, books, etc
5. Host various experiments (games, music stuff, etc)
